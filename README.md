## 依赖

本项目使用 git submodule 管理依赖：

- [prettycli](https://github.com/xshii/prettycli) - CLI 美化工具

## 快速开始

### 克隆项目

```bash
git clone <repository-url>
cd floatshare
git submodule update --init --recursive
```

### 更新子模块

```bash
git submodule update --remote
```
